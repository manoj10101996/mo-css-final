Margin
The margin CSS shorthand property sets the margin area on all four sides of an element.
This property is a shorthand for the following CSS properties:
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
Mo Class for setting margin values in px
  • Margin Top: mt-value
  • Margin Bottom: mb-value
  • Margin Left: ml-value , ms-value
  • Margin Right: mr-value , me-value
  • Margin Horizontal: mx-value
  • Margin Vertical: my-value
