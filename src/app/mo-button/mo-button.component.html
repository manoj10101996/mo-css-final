Button
Always specify the type attribute for a <button> element, to tell browsers what type of button it is.
The <button> HTML element is an interactive element activated by a user with a mouse, keyboard, finger, voice command, or other assistive technology. Once activated,
it then performs a programmable action, such as submitting a form or opening a dialog.
To create a button, just add the .mo-btn class to an inline element like <button> or <a> element.
