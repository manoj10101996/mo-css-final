Alerts
Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.
Alerts are available for any length of text, as well as an optional dismiss button. For proper styling, use one of the required contextual classes
(e.g., .mo-alert .mo-alert-{{color}}).
The notification is a simple colored block meant to draw the attention to the user about something.
To create a alert, just add the .mo-alert class to an inline element like <div> or <p> element.
