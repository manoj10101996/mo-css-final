Padding
The padding CSS shorthand property sets the padding area on all four sides of an element.
This property is a shorthand for the following CSS properties:
  • padding-top
  • padding-right
  • padding-bottom
  • padding-left
Mo Class for setting padding values in px
  • Padding Top: pt-value
  • Padding Bottom: pb-value
  • Padding Left: pl-value , ps-value
  • Padding Right: pr-value , pe-value
  • Padding Horizontal: px-value
  • Padding Vertical: py-value
