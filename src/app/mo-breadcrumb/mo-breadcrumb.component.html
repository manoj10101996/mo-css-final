Breadcrumb
Easily create nice looking notification badges.
The Breadcrumb component consists of links, which are aligned side by side and separated by a divider. Add the .mo-breadcrumb class to a <ul> element to define the component. Use <a> elements as breadcrumb items within the list. An active state is automatically applied to the last <ul> element.
